# Aggregate Result Summaries

This directory contains aggregate result summaries.

## Included

- `four_bridge_positive_meta_summary.json`: a compact aggregate summary of a
  four-bridge positive meta-analysis. It contains only
  aggregate rates/deltas and confidence intervals, not raw code-review comments,
  repository rows, model prompts, or per-example predictions.

This public artifact stays small: enough to show the evidence shape, not enough
to create licensing or privacy ambiguity.
